### 😎 Project Features

- [x] 색상을 보여주는 메인 페이지 (홈)   
   - [x] png 다운로드   
- [x] Palette를 생성할 수 있는 페이지 (등록)   
   - [x] 기본 색상 Color Picker   
   - [x] Neutral Color Picker (검정 <-> 흰색)   
   - [x] 비슷한 색상을 보여주기   
   - [x] 선택한 색상의 opacity 보여주기   
   - [x] Palette 이름 생성하기   
   - [x] 초기화, 선택, 저장 기능   
   - [x] 선택한 색상 변경하기
